About This Project

The X Quotes Generator is a simple web application that allows users to generate screenshots of Twitter (X) posts. It uses the Twitter API to fetch tweet data and Puppeteer to generate screenshots.

This project was created by Isaac Talb as a fun and practical way to learn about web development, APIs, and automation.

Developed by Isaac Talb

Usages: Twitter API and Puppeteer
